# Testing Facility

This is a project to practice my web development skills. A "testing facility" for me to try out different ideas with HTML, CSS, and JavaScript.

## Features

I've added a few things to this page:

* **Color Chooser:** You can click the buttons to change the active color display.
* **Number Counter:** A simple counter that goes from 0 to 10.
* **Date Display & Countdown:** You can input a specific date and time to see it displayed, or start a real-time countdown to that moment. It even calculates leap years for February!
* **Task List:** A to-do list where you can add tasks, mark them as done, or delete them when you're finished.

## How it works

* **HTML/CSS:** Used for the layout and the "Oswald" and "Ubuntu Mono" fonts from Google.
* **JavaScript:** I’m using `setInterval` for the clock and `addEventListener` to catch the Enter key for tasks.
* **Responsive:** I added a viewport tag so it looks okay on the phone too.
